# lp2026 - 石川健二 ポートフォリオサイト

石川健二のプロモーション用ランディングページです。

## 概要

このサイトは、ソフトウェア開発者としての経歴、スキル、プロジェクトを紹介するための個人ポートフォリオサイトです。

## 機能

- **レスポンシブデザイン**: モバイル、タブレット、デスクトップに対応
- **モダンなUI**: クリーンで視覚的に魅力的なデザイン
- **セクション**:
  - ホーム: インパクトのあるヒーローセクション
  - プロフィール: 自己紹介
  - スキル: 技術スタックの紹介
  - プロジェクト: 主要なプロジェクトの展示
  - お問い合わせ: 連絡先情報

## 使い方

### ローカルで確認する場合

1. リポジトリをクローン:
```bash
git clone https://github.com/oqzl/lp2026.git
```

2. ブラウザで `index.html` を開く

### GitHub Pagesで公開する場合

このリポジトリはGitHub Pagesで自動的にデプロイされるように設定されています。

#### 自動デプロイ

- `main` ブランチにプッシュすると、GitHub Actionsが自動的にサイトをビルドしてGitHub Pagesにデプロイします
- デプロイの進行状況は、リポジトリの「Actions」タブで確認できます

#### 初回セットアップ（リポジトリ管理者向け）

GitHub Pagesを有効にするには、以下の手順を実行してください：

1. リポジトリの「Settings」タブを開く
2. 左サイドバーの「Pages」をクリック
3. 「Source」セクションで「GitHub Actions」を選択
4. `main` ブランチにプッシュすると、自動的にデプロイが開始されます

デプロイ後、サイトは `https://oqzl.github.io/lp2026/` でアクセスできるようになります。

## カスタマイズ

`index.html` と `styles.css` を編集して、内容やデザインをカスタマイズできます。

### カスタマイズが推奨される項目:
- **メールアドレス**: `index.html` の `contact@example.com` を実際のメールアドレスに変更してください
- **プロジェクト内容**: プロジェクトセクションの内容を実際のプロジェクトに更新してください
- **自己紹介文**: プロフィールセクションの内容を自分の経歴や目標に合わせて編集してください
- **スキル**: スキルセクションを実際の技術スタックに合わせて調整してください
