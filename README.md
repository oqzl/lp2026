# lp2026 - 石川健治 ポートフォリオサイト

石川健治のプロモーション用ランディングページです。

## 概要

このサイトは、ソフトウェア開発者としての経歴、スキル、プロジェクトを紹介するための個人ポートフォリオサイトです。

## 機能

- **レスポンシブデザイン**: モバイル、タブレット、デスクトップに対応
- **モダンなUI**: クリーンで視覚的に魅力的なデザイン
- **セクション**:
  - ホーム: インパクトのあるヒーローセクション
  - プロフィール: 自己紹介
  - スキル: 技術スタックの紹介
  - プロジェクト: 主要なプロジェクトの展示
  - お問い合わせ: 連絡先情報

## 使い方

1. リポジトリをクローン:
```bash
git clone https://github.com/oqzl/lp2026.git
```

2. ブラウザで `index.html` を開く

または、GitHub Pagesで公開することもできます。

## 技術スタック

- HTML5
- CSS3
- バニラJavaScript (将来的に追加予定)

## カスタマイズ

`index.html` と `styles.css` を編集して、内容やデザインをカスタマイズできます。
